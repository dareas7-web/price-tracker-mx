
# Price Tracker MX 🇲🇽 📉

Bot en Python que rastrea precios de productos en Mercado Libre y Amazon México. Envía alertas a Telegram cuando el precio baja del objetivo.

**Stack:** Python | BeautifulSoup | Requests | Telegram Bot API | SQLite

### Demo
[imagen de tu bot mandando alerta] ← luego la subes

### Características
- ✅ Scraping de Mercado Libre y Amazon MX
- ✅ Base de datos SQLite para historial de precios
- ✅ Notificaciones automáticas por Telegram
- ✅ Corre cada hora con GitHub Actions
- ✅ 100% gratis de hostear

### Instalación
```bash
git clone https://github.com/dareas7-web/price-tracker-mx
cd price-tracker-mx
pip install -r requirements.txt
python main.py
